## Guides

- [Open Source Guide](https://opensource.guide/) by GitHub: A comprehensive collection of resources for individuals, communities, and companies that want to run or contribute to open source projects.
- [Contributor Ladder Template](https://github.com/cncf/project-template/blob/main/CONTRIBUTOR_LADDER.md) by CNCF: A template outlining different contributor roles within a project, along with their responsibilities and privileges.
- [Maintainer Community](https://maintainers.github.com/auth/signin) by GitHub: A private space for maintainers to connect with peers, preview GitHub features, and support the broader open source community.
